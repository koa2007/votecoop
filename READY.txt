Готово к показу

Приложение VoteCoop для голосования ОСГ/кооперативов готово к демонстрации.

Статус: MVP HTML-прототип
Дата: 2026-03-02

Реализовано:
✅ Аутентификация (вход через Google - UI)
✅ Профиль пользователя с полями
✅ Создание групп с 6-значным ID
✅ Присоединение к группам по ID
✅ Создание голосований (обычное, тайное, смена админа)
✅ Голосование за/против
✅ Просмотр результатов
✅ Управление участниками (одобрение запросов)
✅ Система уведомлений
✅ Нижняя навигация (4 экрана)
✅ Mobile-first адаптивная вёрстка
✅ Mock-данные для демонстрации

Известные ограничения (не блокируют демо):
- Primary color синий (#007aff) вместо чёрного (#1A1A1A) по дизайн-системе
- Только украинский язык (без переключателя EN/UK)
- Google Sheets экспорт не реализован
- Нет специальных UI для голосований за добавление/удаление участников

Файлы:
- index.html - основной файл приложения
- css/style.css - стили
- js/app.js - логика приложения
- design-system.md - дизайн-система
- review.md - QA отчёт с полным списком багов/улучшений

Для запуска: открыть index.html в браузере
