# Plan: Приложение для голосования

## Этап 1: HTML Прототип (2-3 дня)

### Sub-agent 1: UX Research & Structure
- [ ] Создать структуру страниц
- [ ] Определить user flows
- [ ] Создать wireframes

### Sub-agent 2: UI Design
- [ ] Разработать дизайн-систему (цвета, типографика, компоненты)
- [ ] Создать макеты всех экранов
- [ ] Экспортировать assets

### Sub-agent 3: Frontend HTML/CSS
- [ ] Создать HTML-структуру
- [ ] Реализовать CSS-стили (минималистичный, светло-серый)
- [ ] Адаптивная верстка (mobile-first)
- [ ] Интерактивность на JavaScript

### Sub-agent 4: Review & Polish
- [ ] Проверить все user flows
- [ ] Тестирование на разных устройствах
- [ ] Финальные правки

## Доставка этапа 1
- Полностью функционирующий HTML-прототип
- Открывается в браузере, работает как мобильное приложение
- Все экраны кликабельны (даже если без бэкенда — мок-данные)

## Этап 2: Backend + PWA (после утверждения)
- Firebase/Supabase для данных
- PWA для установки на телефон
- Push-уведомления

## Этап 3: Native Mobile App
- React Native или Flutter
- Публикация в Play Store
