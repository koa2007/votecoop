# VoteCoop - Приложение для голосования ОСГ/Кооперативов

## 📋 Общее описание проекта

**VoteCoop** — мобильное веб-приложение для проведения голосований в объединениях совладельцев (ОСГ), жилищных кооперативах и других организациях с демократическим управлением.

### Целевая аудитория
- Председатели ОСГ и кооперативов
- Члены жилищных кооперативов
- Управляющие многоквартирных домов

### Основной функционал
1. **Группы** — создание и управление группами (домами/кооперативами)
2. **Голосования** — создание и участие в голосованиях
3. **Участники** — управление членами группы
4. **Уведомления** — информирование о событиях

## ✅ Реализованный функционал

### Экраны приложения
1. **Аутентификация** — вход через Google (UI)
2. **Профиль** — редактирование данных пользователя
3. **Группы** — список групп, создание, присоединение
4. **Детали группы** — участники, статистика, управление
5. **Голосования** — список активных и завершённых
6. **Детали голосования** — участие, результаты, комментарии
7. **Уведомления** — система уведомлений
8. **Инструкции** — справочная информация

### Типы голосований
1. **Обычное** — за/против
2. **Тайное** — скрытые голоса до завершения
3. **Смена администратора** — с фиксированным сроком 72 часа
4. **Удаление участника** — с указанием причины
5. **Заморозка участников** — новый тип (бета)

### Функции участников
- ✅ Сортировка по имени (А-Я / Я-А)
- ✅ Сортировка по участию в голосованиях
- ✅ Поиск по ФИО и телефону
- ✅ Отображение участия (X/Y голосований)
- ✅ Статистика группы (участники, голосования)

### Дополнительные функции
- ✅ Три языка интерфейса (UK, EN, RU)
- ✅ Обязательное поле "Квартира/Участок"
- ✅ Terms of Service при первой регистрации
- ✅ Экспорт истории в CSV
- ✅ Комментарии к голосованиям (500 символов)
- ✅ Воздержание (abstain) с комментарием
- ✅ Удаление голосования автором с причиной
- ✅ Ограничение 1 голосование/24ч для не-админов

## 📝 Для продолжения разработки

### Критические файлы
1. `js/app.js` — вся логика приложения (~3000 строк)
2. `index.html` — все экраны приложения
3. `css/style.css` — стили

### Ключевые объекты состояния (state)
```javascript
this.state = {
    user: { id, firstName, lastName, apartment, phone, frozen },
    groups: [{ id, name, members: [], isAdmin, ... }],
    votings: [{ id, title, type, status, yesVotes, noVotes, ... }],
    notifications: [],
    currentLanguage: 'uk',
    // ... и другие
};
```

### Функции для доработки
- `createVoting()` — создание голосования
- `vote()` — голосование
- `showVotingDetail()` — просмотр деталей
- `renderMembersList()` — отображение участников
- `completeVoting()` — завершение голосования

### Mock-данные
Все данные хранятся в `loadMockData()` — можно изменять для тестирования.
