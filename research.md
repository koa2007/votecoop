# Research: Приложение для голосования ОСГ/кооперативов

## Целевая аудитория
- Жители многоквартирных домов (ОСГ)
- Члены дачных кооперативов
- Небольшие организации, нуждающиеся в демократическом принятии решений

## Core Features (MVP)

### 1. Аутентификация
- Google OAuth (Gmail интеграция)
- Профиль: имя, фамилия, адрес объекта, номер дома/квартиры, телефон

### 2. Группы
- Создание группы с названием ("Дом 61", "СТ Ромашка")
- 6-значный ID группы
- Администратор — создатель группы

### 3. Управление участниками
- Запрос на вступление → одобрение админом
- Голосование за добавление участника вопреки админу (50%+1)
- Голосование за удаление участника (50%+1)

### 4. Голосования
**Типы:**
- Обычное голосование (за/против)
- Тайное голосование (без детализации кто за что)
- Смена администратора (50%+1, 72 часа)
- Удаление группы (50%+1)

**Параметры:**
- Длительность: 1 час — 5 дней
- Ссылка на материалы (Google Drive)
- Цветовая маркировка: 🟡 безрезультатное, 🟢 принято, 🔴 отклонено

### 5. Архивирование
- Google Sheets экспорт истории голосований

### 6. UI/UX
- Минималистичный дизайн
- Светло-серые тоны
- Нижняя навигация: 4 иконки (Голосования, Группы, Уведомления, Профиль)
- Языки: EN / UK

## Технический стек (предварительно)
- Frontend: HTML5 + CSS3 + JavaScript (этап 1)
- Backend: Firebase / Supabase (этап 2)
- Mobile: React Native / Flutter (этап 3)
