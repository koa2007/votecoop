## 2. User Flows

### 2.1 Создание группы

```
Главный экран → Группы (список) → Создание группы
                                      ↓
                            Ввод названия (напр. "Дом 61")
                                      ↓
                           Подтверждение + показ ID (A1B2C3)
                                      ↓
                               Группа создана!
                                      ↓
                            Экран группы (детали)
```

**Детали:**
1. Пользователь нажимает иконку "Группы" в нижней навигации
2. Нажимает кнопку "+" или "Создать группу"
3. Вводит название группы (например, "Дом 61" или "СТ Ромашка")
4. Система генерирует 6-значный ID (например, "A1B2C3")
5. Показывает экран подтверждения с ID для приглашения участников
6. Пользователь становится администратором группы

---

### 2.2 Голосование

```
Уведомление (push) → Детали голосования ← Голосования (список)
                             ↓
                    Просмотр описания, материалов
                             ↓
                       Кнопка "Голосовать"
                             ↓
                    Выбор варианта (За/Против/Воздержаться)
                             ↓
                    Тайное? → Да → Анонимный выбор
                    Нет → Подтверждение выбора
                             ↓
                        Голосование учтено ✓
                             ↓
                    Детали (обновленные результаты)
```

**Детали:**
1. Пользователь получает push-уведомление о новом голосовании ИЛИ открывает список
2. Просматривает детали: название, описание, ссылку на материалы, сроки
3. Нажимает "Голосовать"
4. Выбирает вариант ("За" или "Против")
5. Если голосование тайное — выбор анонимизирован
6. Подтверждает выбор
7. Видит подтверждение и обновлённые результаты (если не тайное)

---

### 2.3 Смена администратора

```
Группа (детали) → Меню ⋮ → Настройки группы
                                    ↓
                          "Сменить администратора"
                                    ↓
                            Выбор участника
                                    ↓
                          Проверка прав (50%+1)
                          ↓ Да              Нет ↓
                    Создание голосования  Ошибка прав
                          ↓
                    72 часа голосования
                          ↓
              50%+1 голосов
              ↓ Да              Нет ↓
        Новый админ назначен  Админ не сменился
```

**Детали:**
1. Текущий администратор открывает детали группы
2. Переходит в "Настройки группы" через меню
3. Выбирает "Сменить администратора"
4. Выбирает участника из списка
5. Система проверяет наличие прав (инициатор должен иметь 50%+1 голосов участников)
6. Создаётся голосование на 72 часа
7. Все участники получают уведомление
8. По истечении 72 часов подсчитываются результаты
9. Если 50%+1 "За" — администратор меняется
10. Все участники получают уведомление о результате

---

### 2.4 Вступление в группу

```
Группы (список) → + Вступить → Ввод ID (6 символов)
                                         ↓
                                  Проверка группы
                                  ↓ Да              Нет ↓
                            Показ информации      Ошибка (не найдено)
                                  ↓
                          Отправить запрос
                          ↓ Открытая      Закрытая ↓
                    Автоматическое       Запрос отправлен
                    добавление                  ↓
                          ↓            Админ одобрил
                       В группе!               ↓
                                          В группе!
```

## 4. Состояния и ошибки

### 4.1 Состояния загрузки
- Скелетоны при загрузке списков
- Индикаторы загрузки на кнопках
- Pull-to-refresh для обновления

### 4.2 Пустые состояния
- Нет активных голосований: "Пока нет голосований. Создайте первое!"
- Нет групп: "Вы еще не вступили ни в одну группу"
- Нет уведомлений: "Уведомлений пока нет"

### 4.3 Состояния ошибок
- Нет соединения: "Проверьте подключение к интернету"
- Ошибка сервера: "Что-то пошло не так. Попробуйте позже"
- Неверный ID группы: "Группа не найдена. Проверьте ID"
- Недостаточно прав: "Только администратор может выполнить это действие"

## 5. Дополнительные сценарии

### 5.1 Голосование за добавление/удаление участника
- Инициируется при отказе админа принять запрос
- Требует 50%+1 голосов участников группы
- Продолжительность: 72 часа
- Тип: обычное (не тайное) для прозрачности

### 5.2 Удаление группы
- Требует 50%+1 голосов
- Длительность: до 5 дней
- После удаления: архив сохраняется 30 дней, потом удаляется

### 5.3 Экспорт в Google Sheets
- Доступен только администратору
- Экспортирует историю всех голосований группы
- Формат: дата, название, результаты (за/против/воздержались)
